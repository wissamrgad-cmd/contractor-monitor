Data & sync — GitHub Gist
Automatic sync via GitHub Gist
Your data is stored as a private file on GitHub. The app automatically loads the latest data every time you open it, and saves every change. Works on PC and Android. No library or CDN required.
1 Create a free GitHub account (skip if you have one)
Go to github.com and sign up for a free account. It takes 2 minutes.
2 Generate a Personal Access Token
  1. Sign in to GitHub and click your profile photo (top right) → Settings
  2. Scroll to the bottom of the left sidebar → click Developer settings
  3. Click Personal access tokens → Tokens (classic)
  4. Click Generate new token (classic)
  5. Give it any name, set expiry to No expiration
  6. Under Select scopes, check only gist
  7. Click Generate token — copy the token shown (you only see it once)
3 Connect and start syncing
First device (PC): Leave the Gist ID field empty and click Connect — a new Gist is created automatically.
Second device (mobile): Copy the Gist ID shown after connecting on PC and paste it here before clicking Connect.
